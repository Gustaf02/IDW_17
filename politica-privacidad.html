Política de Privacidad

Última actualización: 3 de Noviembre, 2025

Clínica Salud Integral ("nosotros", "nuestro") opera el sitio web (el "Servicio"). Esta página le informa sobre nuestras políticas con respecto a la recopilación, uso y divulgación de datos personales cuando utiliza nuestro Servicio.

1. Recopilación y Uso de Información

Recopilamos varios tipos diferentes de información para diversos fines para proporcionar y mejorar nuestro Servicio para usted. Al reservar un turno, le pediremos que nos proporcione cierta información de identificación personal que puede incluir, entre otros, su nombre completo y número de documento.

2. Seguridad de los Datos

La seguridad de sus datos es importante para nosotros, pero recuerde que ningún método de transmisión por Internet o método de almacenamiento electrónico es 100% seguro. Si bien nos esforzamos por utilizar medios comercialmente aceptables para proteger sus Datos personales, no podemos garantizar su seguridad absoluta.

3. Uso de Datos

Clínica Salud Integral utiliza los datos recopilados para diversos fines:

  • Para gestionar sus reservas de turnos.
  • Para notificarle sobre cambios en nuestro Servicio.
  • Para proporcionar atención y soporte al paciente.
  • Para cumplir con cualquier fin legal.
